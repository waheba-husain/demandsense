# demandsense
DemandSense is a data analysis project that studies electricity retail sales demand using the EIA API.
## Features
- Fetches electricity retail sales data using API
- Data cleaning and analysis with Pandas
- Demand trend exploration
## Tech Stack
- Python
- Pandas
- Requests
- Google Colab
## How to Run
1. Open the notebook in Google Colab
2. Add your API key using Colab Secrets
3. Run all cells
